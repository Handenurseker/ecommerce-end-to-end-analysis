# E-Commerce Sales Analysis
This project analyzes e-commerce sales data using SQL and Power BI to evaluate revenue performance, city-based sales distribution, customer behavior, and category-level sales trends.

## Project Overview
This project was created as an end-to-end sales analytics case study. It combines SQL-based analysis with Power BI dashboards to identify business insights from e-commerce sales data.

## Business Problem
This project aims to analyze e-commerce sales data by examining total revenue, city-level performance, category-based sales distribution, and customer behavior. 
Using SQL and Power BI, key performance indicators were analyzed, high-performing cities and customers were identified, and visual reports were created to support business decision-making.

## Analysis Questions
- What is the total revenue generated from e-commerce sales?
- Which cities generate the highest revenue?
- How do cities contribute to total revenue?
- What is the average revenue per customer?
- Which cities perform above average in sales?
- How is revenue distributed across product categories?
- Which customers generate the highest revenue?
- What is the repeat customer rate?
  
## Tools Used
- SQL Server
- Power BI
- Excel / CSV
- Power Query
  
## Key Insights
- Total revenue is approximately **₺13.2M**.
- The total number of customers is approximately **9,678**.
- Average revenue per customer is around **₺1.36K**.
- **Istanbul** is the highest revenue-generating city, with approximately **₺1.31M** in revenue.
- Istanbul contributes **9.89%** of total revenue.
- Average city revenue is approximately **₺163.0K**.
- Cities performing above average revenue include **Istanbul, İzmir, Ankara, Bursa, and Adana**.
- The highest revenue-generating category is **Home**.
- The second highest revenue-generating category is **Toys**.
- Female customers represent **53.6%** of the customer base, while male customers account for **46.4%**.

## Business Recommendations
- Focus marketing efforts on top-performing cities to maximize revenue growth.
- More targeted campaigns can be planned for cities performing above average revenue.
- Product and campaign strategies can be strengthened for high-performing categories such as **Home** and **Toys**.
- Customer retention strategies can be developed for top revenue-generating customers.
- Additional campaigns focused on customer acquisition can be considered to attract more new customers.
